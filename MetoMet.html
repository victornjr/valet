Objetivo Estratégico

Ahorrar tiempo - buscar máquina para pagar ticket de estacionamiento, hacer filas, buscar cambio, esperar por asistencia técnica -.

Tareas

  • Registrarse
  • Pagar

Pasos para registrarse:

1 Opción uno. Te manda a Facebook para que permitas que la aplicación tenga acceso a tu perfil y te notifica que no va a publicar nada en tu nombre.

2Opción dos.Te pide ingresar correo electrónico, celular y contraseña, después que escribas tu nombre, apellido y subas tu foto.

3Escoger entre las formas de pago disponibles: tarjeta de crédito, tarjeta de débito o Paypal. Puedes escanearla con cámara o ingresar los dígitos.

4La aplicación enviará un mensaje de texto al celular, el cual contiene el código para finalizar el registro.

Pasos para pagar:

1Escanear el código QR.

2La aplicación visualizará el tiempo que estuviste y la suma a pagar.

3Aceptar que se realice la transacción.

4La aplicación notificará que el usuario tiene 15 minutos para salir.
